# CopilotKit Playground

## Getting Started

### Prerequisites

- Node.js installed
- Google API Key

### Setup

1. Create a `.env` file in the root of the project and add your Google API Key:
   ```env
   GOOGLE_API_KEY=your_api_key_here


To run the frontend:

```
npm start
```

To run the Node server:

```
npm run server
```
